# Power Apps Canvas App – Source Code

This repository contains the **unpacked source code** for a Microsoft **Power Apps Canvas App**.  
The app is stored in a **Git-friendly source format** and can be packed back into a `.msapp` file using the **Power Platform CLI**.

---

## Repository Structure

```text
/
├── Src/                 # Screens and Power Fx logic (.fx.yaml)
├── Controls/            # Custom controls
├── Media/               # Images, icons, and other assets
├── Connections/         # Connection references (no secrets)
├── app.json             # App metadata
├── CanvasManifest.json  # Canvas app manifest
└── README.md
``
